# Student Management System

## Overview

A web-based Student Management System developed using ASP.NET Core MVC, Entity Framework Core, and SQL Server.

The application allows users to manage student records through Create, Read, Update, and Delete (CRUD) operations.

## Features

* Add Student
* View Students
* Edit Student Details
* Delete Student Records
* Search Students
* Form Validation
* Responsive Bootstrap UI
* Dashboard Statistics
* SQL Server Database Integration

## Technologies Used

* ASP.NET Core MVC
* C#
* Entity Framework Core
* SQL Server
* Bootstrap 5
* Visual Studio Code

## Project Structure

* Models
* Views
* Controllers
* Data
* Migrations

## Database

Entity Framework Core Code First approach was used to create and manage the SQL Server database.

## How to Run

1. Clone the repository

git clone https://github.com/phanikumar124/Student-Management-System-ASP.NET-Core-MVC.git

2. Open project folder

3. Update connection string in appsettings.json

4. Apply migrations

dotnet ef database update

5. Run application

dotnet run
